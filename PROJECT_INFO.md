# Work Tracker Pro - Техническая документация

## 🏗 Архитектура
Приложение построено как **SPA (Single Page Application)** с использованием современных инструментов веб-разработки. Основной упор сделан на производительность, плавность интерфейса и локальное хранение данных.

- **Frontend:** React 18 + Vite
- **Стилизация:** Tailwind CSS (Utility-first)
- **Анимации:** Framer Motion (библиотека `motion/react`)
- **База данных:** IndexedDB (через кастомную обертку для локального хранения)
- **Графики:** Recharts
- **Экспорт:** jsPDF + jspdf-autotable

---

## 🎨 Дизайн-система

### Палитра цветов (CSS Variables)
Используется динамическая тема, переключаемая через класс на `html`.
- `--bg`: Основной фон (светлый/темный)
- `--bg-1`: Фон карточек и элементов
- `--t1`: Основной текст (высокий контраст)
- `--t2`: Вторичный текст
- `--t3`: Приглушенный текст / подписи
- `--a`: Акцентный цвет (Золотисто-желтый `#C8B560`)
- `--b`: Цвет границ и разделителей

### Шрифты
- **Основной:** `Inter` (Sans-serif) — для интерфейса и читаемости.
- **Цифры и заголовки:** `Outfit` или системный `Sans` с жирным начертанием (`font-black`).

---

## 🛠 Frontend & UI
- **Компоненты:** Функциональные компоненты React с хуками (`useState`, `useMemo`, `useEffect`).
- **Иконки:** `Lucide React` (используются как база для анимированных SVG).
- **Адаптивность:** Mobile-first дизайн, оптимизированный под использование в качестве PWA.

---

## ⚙️ Backend & Данные
- **Локальное хранение:** Все данные (записи времени, настройки) хранятся в браузере пользователя через **IndexedDB**. Это обеспечивает работу без интернета и мгновенный доступ.
- **Синхронизация:** Реализована логика "Sync", имитирующая облачную синхронизацию (подготовлена база для интеграции с внешним API).

---

## ✨ Анимации (Framer Motion Secrets)
Анимации — ключевая фишка приложения, создающая ощущение "премиальности".

- **Navigation:** Используется `layoutId="nav-indicator"` для плавного перемещения подчеркивания между пунктами меню.
- **Micro-interactions:** 
  - Иконки в таббаре имеют уникальные физические анимации (Spring) при клике.
  - Кнопки используют `whileTap={{ scale: 0.95 }}`.
- **Transitions:** Плавное появление экранов через `AnimatePresence` и `initial={{ opacity: 0, y: 10 }}`.
- **Charts:** Графики Recharts анимируются при загрузке, а круговой прогресс использует `strokeDashoffset` с `transition: { duration: 1.5, ease: "easeOut" }`.

---

## 🚀 Основные фичи
1. **Трекер времени:** Секундомер в реальном времени с сохранением состояния.
2. **Аналитика:** 6-месячный тренд доходов и визуализация достижения целей.
3. **Управление финансами:** Настройка почасовой ставки, налогов, бонусов и сверхурочных.
4. **Экспорт отчетов:** Генерация детализированных PDF и CSV файлов.
5. **Privacy Mode:** Возможность скрыть финансовые данные одним кликом (глаз в шапке).
6. **Мультиязычность:** Поддержка русского и английского языков.

## 📝 Заметки разработчику
- При добавлении новых SVG-анимаций всегда используйте числовые значения для атрибутов (`cx={10}`, а не `cx="10"`), чтобы избежать ошибок интерполяции в Framer Motion.
- Всегда проверяйте `goalPct` на `NaN` или `undefined` перед передачей в SVG-атрибуты.
